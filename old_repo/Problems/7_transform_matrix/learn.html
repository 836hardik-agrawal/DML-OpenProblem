Matrix Transformation using \(T^{-1}AS\)

Transforming a matrix \(A\) using the operation \(T^{-1}AS\) involves several steps. This operation changes the basis of matrix \(A\) using two matrices \(T\) and \(S\), with \(T\) being invertible. Given matrices \(A\), \(T\), and \(S\):

1. Check if \(T\) is invertible by ensuring it's determinant is non-zero; else return -1.

2. Compute the inverse of \(T\), denoted as \(T^{-1}\).

3. Perform the matrix multiplication to obtain the transformed matrix:

\[ A' = T^{-1}AS \]
